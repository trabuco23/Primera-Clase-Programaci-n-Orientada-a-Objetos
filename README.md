# Primera-Clase-Programaci-n-Orientada-a-Objetos
3 problemas sencillos de prueba

**Resumen – Primera clase de Programación Orientada a Objetos**

En nuestra primera clase de Programación Orientada a Objetos no abordamos todavía los conceptos propios del curso, sino que realizamos ejercicios de repaso con el fin de verificar si aún manteníamos las destrezas adquiridas en cursos anteriores. Estos ejercicios nos permitieron medir nuestro nivel actual y recordar estructuras básicas de programación, sintaxis y lógica que serán necesarias como base para continuar.

**Objetivos de la sesión:**

* Reforzar conocimientos previos en programación.
* Identificar fortalezas y vacíos en el manejo de estructuras básicas.
* Prepararnos para la introducción de los conceptos fundamentales de POO.

**Observaciones:**
La clase fue principalmente práctica, lo que facilitó retomar la lógica de programación después de las vacaciones. Averiguamos que quiza que algunos temas requieren repasarse con mayor detalle, lo cual servirá de punto de partida para avanzar de manera más sólida en el curso.
